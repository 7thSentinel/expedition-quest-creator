# 427: unknown key: <key>
## Details:

The quest metadata key you're trying to use must be a number.
## Incorrect:

```markdown
# Test Quest
summary: A quest that'll test ya
author: Test McTesterson
minplayers: 1
maxplayers: 6
nonsense: 1
```

## Correct:

```markdown
# Test Quest
summary: A quest that'll test ya
author: Test McTesterson
minplayers: 1
maxplayers: 6
```
